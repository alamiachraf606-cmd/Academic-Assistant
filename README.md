# Decentralised Academic Assistant

This repo scaffolds the course project. The AI/RAG service is intentionally left
out for now and will be added after the core blockchain, backend, and frontend
are working.

## Structure
- chain/: Hardhat project with Solidity contracts + tests (viem).
- backend/: Express API (placeholders for now).
- frontend/: Student + Professor web UIs (HTML/CSS/JS).
